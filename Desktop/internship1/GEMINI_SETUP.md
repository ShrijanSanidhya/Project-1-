# How to Get a Google Gemini API Key

1.  **Go to Google AI Studio**: Visit [https://aistudio.google.com/](https://aistudio.google.com/).
2.  **Sign In**: Log in with your Google account.
3.  **Get Key**:
    *   On the left sidebar, click on **"Get API key"**.
    *   Click the **"Create API key"** button.
    *   Select **"Create API key in new project"** (or an existing one).
4.  **Copy**: Copy the generated key string (starts with `AIza...`).
5.  **Save**: Paste this key into your `backend/.env` file as `GEMINI_API_KEY`.
